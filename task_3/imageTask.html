4 other psuedo classes and 4 other pseudo elemetns that I never used before

psuedo classes

1.:nth-of-type(n): This pseudo-class allows you to select elements based on their position within their parent.
You can specify a numerical value n to select every nth element of a certain type. For example, p:nth-of-type(odd) selects all odd-numbered paragraphs

2.:not(selector): The :not() pseudo-class allows you to select elements that do not match a specific selector. For example,
a:not(.external) selects all anchor links that do not have the class "external."

3.:focus-within: This pseudo-class selects an element when any of its descendants receive focus. It's commonly used in forms and
interactive elements to style parent elements when child elements have focus.

4. :empty: The :empty pseudo-class selects elements that have no children or contain only whitespace. It's useful for styling
or targeting elements based on their content.

psuedo elements

1.::before: The ::before pseudo-element is used to insert content before the content of an element. It's often employed to add
decorative elements or labels before an element's content.

2.::after: Similar to ::before, ::after inserts content after the content of an element. It's frequently used to add decorative
elements or generate content dynamically.

3.::first-line: The ::first-line pseudo-element selects the first line of a block-level element. You can apply styles specifically to the
first line of text within a paragraph or heading, for example..

4. ::first-letter: The ::first-letter pseudo-element selects the first letter or character of a block-level element. It's often used to apply
styles to initial characters for drop caps or other decorative effects..

What does box-sizing and border do

box-sizing

It makes an element calculate its total size, including its content, padding, and borders.
This means that when you set the width or height of the element, it includes the space taken up by padding and borders, making it easier to create predictable and responsive layouts

border box

It's a CSS property used to define the border of an element.
You can specify the border's width, style, and color using this property. It creates a visual boundary around an element, helping you define its edges and make it stand out from other content.

2 other ways of using color

Background color

You can set the background color of elements, such as the background of a webpage, divs, sections, or buttons.
Background color can create visual separation, highlight important content, and contribute to the overall design and aesthetics of a website.
body { background-color: #f0f0f0; /* Set the background color of the entire webpage */ }

Text- color

You can set the color of text to improve readability and create visual hierarchy. Different text colors can be used for headings,
body text, links, and more. Choosing appropriate text colors that contrast well with the background helps users read content comfortably.
These are just a couple of ways color is used in web design. It's also important for branding, visual consistency, and conveying emotions and messages through design.
h1 { color: #333; /* Set the text color of heading elements */ } a { color: #0077cc; /* Set the text color of links */ }
